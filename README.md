# 🚀 DesCodes - Solutions Numériques Professionnelles

> **Votre partenaire pour des solutions numériques sur mesure, innovantes et performantes.**

Basé en Indre-et-Loire (37), **DesCodes** accompagne les artisans, entrepreneurs et entreprises dans leur transformation digitale avec des sites vitrines, des applications web et des solutions d'Intelligence Artificielle.

---

## ✨ Fonctionnalités Clés

- 🌓 **Mode Sombre / Clair Intuitif** : Un design moderne avec basculement fluide et synchronisation instantanée entre tous les onglets ouverts.
- 📱 **Design Ultra-Responsive** : Une expérience utilisateur optimale sur tous les supports (mobiles, tablettes, ordinateurs).
- ⚡ **Animations de Défilement** : Des effets d'apparition subtils pour une navigation dynamique et engageante.
- 🎯 **Optimisation SEO** : Structure HTML5 sémantique pour un meilleur référencement naturel.
- 🛠️ **Gestion des Tarifs** : Une page dédiée présentant des packs transparents (Présence, Artisan, Business).

---

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icônes** : [FontAwesome 6.4.0](https://fontawesome.com/)
- **Typographie** : [Inter](https://fonts.google.com/specimen/Inter)
- **Hébergement suggéré** : GitHub Pages, Netlify ou OVH.

## ⚙️ Installation & Utilisation

Le projet est conçu en **Vanilla JS/CSS**, il ne nécessite aucun compilateur ou framework lourd.

1. **Cloner le projet** :
   ```bash
   git clone [URL_DU_DEPOT]
   ```
2. **Lancer le site** :
   Ouvrez simplement `index.html` dans votre navigateur préféré.
   
*Note : Pour une synchronisation parfaite du thème localement, il est recommandé d'utiliser une extension de type **Live Server**.*
